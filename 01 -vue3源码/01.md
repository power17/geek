## 3.搭建源码学习环境
```shell
 git clone git@github.com:vuejs/core.git 
pnpm install
# 在 package.json 上加 --sourcemap
pnpm  dev
# server
pnpm serve
```
#### 打开/packages/vue/examples/composition/todomvc 
- 打开开发者工具source
- cmd+P  打断点 createApp
- 找源文件 右键 reveal in sidebar
