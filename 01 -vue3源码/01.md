- 装13
- 内卷应付面试
- 加深理解
- 增强解决问题能力
- 掌握真正傍身技能
## 2.源码学习方法分享
- 找到一条主线
- 画一张思维导图
- 提出一个问题，去源码中找答案
- 造一个轮子
- 费曼学习法
## 3.搭建源码学习环境
```shell
 git clone git@github.com:vuejs/core.git 
pnpm install
# 在 package.json 上加 --sourcemap
pnpm  dev
# server
pnpm serve
```
#### 打开/packages/vue/examples/composition/todomvc 
- 打开开发者工具source
- cmd+P  打断点 createApp
- 找源文件 右键 reveal in sidebar
